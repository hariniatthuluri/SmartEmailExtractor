## 📧 Smart Email Extractor & Organizer

## 🚀 Overview
A Python automation script that extracts email addresses from a text file, removes duplicates, and saves them in a clean format.

## ✨ Features
- Extracts emails using Regex
- Removes duplicates automatically
- Saves sorted output
- Logs activity with timestamps
- Clean and modular code

## 🛠️ Tech Stack
- Python
- Regex (`re`)
- File Handling
- OS module
